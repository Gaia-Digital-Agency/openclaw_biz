# SKILL-LETTER.md — Letter Writing

Use this skill when asked to write, draft, or send a formal letter.

## Letter Format

```
Dear [Recipient],

[Body content — clear, professional, concise]

Kind regards,
Casey
Business Assistant
```

## Rules

- Use professional, clear language.
- Keep letters concise and purposeful.
- If recipient name is not provided, use "Dear Sir/Madam".
- If asked to email the letter, use the `gmail` MCP tool (`send_email`) with the letter content as the email body.
- If asked to save to Drive, use the `google-drive` MCP tool to upload as a document.
